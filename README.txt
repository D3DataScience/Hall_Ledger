Hall Ledger Project Note

Hall Ledger is an idea I had for a long time before I started building it. The structure of the project, the induction categories, the player choices, the standards behind each wing, and the overall editorial direction all came from me. This site reflects the Hall of Fame project I wanted to create, and I guided its content, organization, and presentation.

I also did a decent amount of the data-based R coding myself. Much of the data cleanup, corrections, testing, review work, and decision-making around how the R code logic should behave came directly from me. I am still developing my R skills, and this project has been a good test of what I already have while also giving me a look at more advanced ideas and workflows.

Codex was used to design and implement all of the front-end for Hall Ledger. That includes the layout, styling, homepage structure, player-page presentation, navigation, and the overall interface work. Codex was also used for a large share of the more technical R work, especially where the project needed more advanced scripting, debugging, automation, or structural changes.

I don't consider Hall Ledger to be a full reflection of my coding ability; however, it is my project and my idea, with Codex serving as the coding and implementation assistant that helped me build it. The editorial decisions are mine.

All comments in the code have been generated by Codex as I have been using them to learn more about HTML, CSS, JS, and new parts of R
